HTML5 and CSS3

This course introduces the latest updates to HTML5 and CSS3. It is intended to be an easy introduction to get up-to-date with the current standard, but not a complete reference. You have the W3C for that.

What we do provide:

  • 600 slides
  • 30 labs
  • 90+ interactive examples embedded in the slides

The course is designed to be delivered in 24 hours including the labs. It is full of interactive examples, but still the slides are deliberately sketchy since we strongly believe that Instructor Matters.

Not tested in all browsers

We are a bunch of lazy bastards. We prefer to introduce the latest crazy-ass technology instead of making the slides work with every browser under the sun. This means that the slides themselves have only been tested to work with Google Chrome. If you have a standards-compliant way of doing things better please pull-request us, but don't waste your time including compatibility with MyBrowser(tm), be it retro, mobile, desktop or DeathStar-compliant because we just don't care.

These slides have been built with reveal.js, a framework for HTML presentations. Learn by using it.
