Charset defaults

  • http default: ISO8859-1
  • multipart/form data default: US ASCII
  • W3C official recommendation: UTF-8

  • HTTP header

GET vs POST

  • GET: idempotent request
    • read-only operations, like retrieving data
  • POST: non-idempotent request
    • writeable operations, like storing or deleting data
  • Other: PUT, HEAD, DELETE

HTML5 obsolete attributes (W3C)

Importance tags

  • Use <b>
    • text without specific emphasis or importance typically represented as bold text
    • keywords, product names, etc.
    • E.g.: Use <b> to indicate bold text.
  • Use <strong>
    • Strong semantic importance
    • E.g.: It is important that you do not mix styles up!

Emphasis tags

  • Use <i>
    • text without emphasis or importance, typically represented as italic text
    • technical term, phrase from another language, a ship name, etc.
    • E.g.: Intoxicated from eating too much wasabi at once.
  • Use <em>
    • Stress emphasis: use different voice or mood
    • E.g.: This is not a place for cool people anymore.
